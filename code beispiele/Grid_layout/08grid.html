Das Grid

Eins

Das grid ist ein zweidimensionales Layout-Tool. Nach der Deklaration eines Grid-Containers können Grid-Items sowohl in Zeile als auch in Spalte angelegt werden. Die Reihenfolge der Elemente im Stylesheet kann vollkommen unabhängig von der Reihenfolge derselben Elemente im HTML-Dokument festgelegt werden.

Grid Vokabeln

  • row
  • column
  • area
  • line

Der Grid-Container

Zwei

Mit display: grid wird der Grid-Container deklariert. Alle direkten Kinder sind Grid-Items und können nun im Grid platziert werden.

Im Grid-Container müssen die rows, columns und/oder die areas festgelegt werden. Erst dann können die Grid-Items platziert werden.

Die Grid-Items

Drei

Die Items können überall im Grid platziert werden, unabhängig von ihrer Position im HTML-Dokument. Werden Sie nicht explizit platziert, dann werden Sie gemäß der Reihenfolge im HTML-Dokument aufgelistet.

Vier

  • grid-column-gap

  • grid-gap
  • justify-items (row)
  • align-items (column)
  • place-items (row and column)
