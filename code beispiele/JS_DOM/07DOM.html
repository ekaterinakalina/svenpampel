js und das DOM

DOM (Document Object Model)

Ein HTML-Dokument ist hierarchisch, wie eine Baumstruktur, aus vielen Knoten aufgebaut. Der oberste oder Wurzelknoten ist das Dokument selbst, das document.

Mit js kann man auf das DOM zugreifen, es auslesen, Knoten löschen, ändern oder hinzufügen.
